Details to Run The Website

1. download XAMPP and install it in directory C:\xampp.
2. Copy this Project Folder to the Location- C:\xampp\htdocs\..
3.Import the Database SQL File "Campus1" in your database i.e. using phpmyadmin.
4.Run XAMPP and start Apache and MySQL.
5. Open any Browser And go to "localhost/myprojectt" 
6.Then you Can Go Through the Website.
